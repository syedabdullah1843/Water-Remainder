# 💧 Water Reminder (Python)

A simple Python desktop application that reminds you to drink water at regular intervals using system notifications.

---

## 🚀 Features

* 💻 Desktop notifications
* ⏱ Customizable reminder interval
* 🔁 Runs continuously in background
* ⚡ Lightweight and easy to use

---

## 🛠 Tech Stack

* Python 3
* plyer (for desktop notifications)

---

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/water-reminder.git
cd water-reminder
```

2. Install dependency:

```bash
pip install plyer
```

---

## ▶️ Usage

Run the script:

```bash
python water_reminder.py
```

---

## ⚙️ Customization

To change reminder interval, edit this line in the code:

```python
time.sleep(100)  # change seconds as needed
```

Examples:

* 600 → 10 minutes
* 1800 → 30 minutes

---

## 🛑 Stop the Program

Press:

```
Ctrl + C
```
